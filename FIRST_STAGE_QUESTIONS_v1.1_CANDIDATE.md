# 第一阶段通用题库 — 候选修改版本 v1.1

> 项目定位：公益专业认知网站，不做心理诊断，不替用户填志愿
> 版本标识：FIRST_STAGE_VERSION = "v1.1"
> 阶段：候选版本（未覆盖正式题库）

---

## 总体判断

**指向性过强程度：严重。** 当前12道题中，多数题目选项几乎等同于"六大专业大类直选题"。高中生做完第一题就能猜到"我选A就是人文、选C就是工科"，降低了测试的真实性和探索感。

**最明显的焦点题（ordered by severity）：**
1. **gen_001** — 最典型的大类直选题。题干："周末空出一整个下午"，选项直接写成"读书写作/聊天助人/捣鼓设备/看展画画/健康自然"
2. **gen_007** — 选项直接用"文字、公式、实验、地理、创作"作为标签
3. **gen_010** — "思考方式"逐项对应六大类，非常刻意
4. **gen_002** — "写文章/组织活动/算账/做装置/恢复健康/创作作品"，六道选项全部对标专业大类
5. **gen_003** — 和第二题高度同质，换了个花钱的包装

**主要风险：**
1. **社会期望偏差**：高中生可能会选"看起来更体面"的选项而非真实偏好
2. **用户失去信任**：太明显的标签会让测试者觉得"这不就是在给我贴标签吗"
3. **无探索感**：选项的可识别性过强，剥夺了用户在测试过程中自我发现的可能性
4. **维度同质化**：gen_001/gen_002/gen_003/gen_007 高度重复（人文→理工→社科→商科→艺术→生命）

---

## 修改原则

### 核心策略

把"你选哪个专业大类"变成"你更喜欢哪种学习和思考方式"，具体做法：

1. **去标签化**：移除选项中的专业关键词（如"读书""画画""装置""病""程序"），换成行为描述
2. **行为到偏好**：让用户选的是一种**认知/行为方式**，而非一个**活动名称**
3. **容许多个方向走同一选项**：设计选项时，1个选项可以同时对应2-3个维度的加分，不要每题严格一一对应
4. **变中性的对标**：例如"把一个复杂想法整理成清楚的表达"可以同时给 humanities(阅读表达) 和 business(商业/管理) 加分

### 加分调整原则

| 内容 | 原范围 | 建议范围 |
|------|--------|----------|
| 大类标签（bucket） | +6 ~ +12 | +6 ~ +10 |
| 二级能力标签 | +4 ~ +8 | +4 ~ +6 |
| 负分 | rule_detail(+-6) | 取消负分或改为明确负分 |
| 每选项标签数 | 1~4个 | 建议1~2个主标签，最多2个辅标签 |

## 需要人工确认的问题

### 1. gen_016 的"不确定性接受度"是否应计分？
**核心争议：** 将"高不确定性接受=人文/艺术"缺乏实证依据。建议改为新增 `exploration_preference` 标签，但会需要在推荐算法中加入对该标签的支持逻辑。
👉 **建议：暂不加到大类计分，只作为风险提示字段。**

### 2. gen_017 的"工作价值观"是否应独立为第二维度？
**核心争议：** gen_017 本质是价值观而非兴趣。将所有价值观映射到专业大类可能造成系统性偏差。
👉 **建议：考虑在后续版本中，将 gen_017 单独拉出作为"价值观匹配层"，与兴趣匹配层并行判断。**

### 3. gen_020 是否应完全移出专业大类计分？
**核心争议：** "如何面对负面评价"是一个性格特征（尽责性/神经质），与专业偏好无关。目前在20题中占据12分权重过高。
👉 **建议：移出大类计分，改为报告页中的"性格提示"段落，而非推荐依据。**

### 4. gen_003 缺少生命健康选项
当前4个选项（人文/社科/理工/艺术），无生命健康对应选项。
👉 **建议：增加第5个选项，或承认这道题不覆盖生命健康。**

### 5. 新增标签提案

| 建议新增标签 | 来源 | 用途 |
|-------------|------|------|
| `exploration_preference` | gen_016 | 衡量对不确定性的接受度，不直接进大类计分 |
| `self_improvement_preference` | gen_017_B | 表示成长驱动力，跨领域通用 |
| `feedback_reflection` | gen_020 | 面对负面评价的反思倾向，不直接进大类计分 |
| `autonomy_preference` | gen_017_E | 对自主性的需求程度，跨领域通用 |

### 6. 整体权重问题
12道固定题中，6道（gen_001/002/003/007/008/010）高度同质，覆盖方向几乎完全相同。实际上12题中只有 ~8题提供了有区分度的信息。
👉 **建议：未来版本考虑替换部分同质化题目。**
